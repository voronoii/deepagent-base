# HUG RAG - 전세/부동산 법령 및 가이드 검색 도구

## 개요

HUG(한국주택금융공사) RAG MCP 도구는 전세계약 관련 법령, HUG 가이드/매뉴얼 문서를 벡터 유사도 검색 + 리랭킹으로 검색하는 도구입니다.
Qdrant 벡터DB 기반이며, `snowflake-arctic-embed-l-v2.0-ko` 임베딩 + `bge-reranker-v2-m3-ko` 리랭커를 사용합니다.

---

## MCP 도구 목록

### 1. `search_hug_docs` — 문서 검색 (핵심 도구)

벡터 유사도 검색 + 리랭킹으로 가장 관련성 높은 문서를 반환합니다.

| 파라미터 | 타입 | 필수 | 설명 |
|---------|------|------|------|
| `query_text` | string | **필수** | 검색 질의. 자연어 문장으로 작성 |
| `domain` | string | 선택 | `"law"` = 법령만, `"guide"` = 가이드/매뉴얼만, `""` = 전체 |
| `title_filter` | string | 선택 | 특정 문서/법령명으로 필터링 |
| `limit` | int | 선택 | 반환 결과 수 (기본값: 5) |

**반환 필드:** 유사도, 도메인, 문서제목, 조문/섹션명, 조문/섹션내용, 카테고리, 출처파일, 법령링크(법령인 경우)

### 2. `list_available_docs` — 저장 문서 목록 조회

현재 DB에 저장된 문서 목록과 도메인별 통계를 확인합니다.

| 파라미터 | 타입 | 필수 | 설명 |
|---------|------|------|------|
| `domain` | string | 선택 | `"law"`, `"guide"`, `""` (전체) |

### 3. `get_doc_by_title` — 특정 문서 조문/섹션 조회

벡터 검색 없이 제목 기반 필터로 정확히 조회합니다. 특정 법령의 조문을 순서대로 읽거나, 특정 가이드의 전체 내용을 확인할 때 사용합니다.

| 파라미터 | 타입 | 필수 | 설명 |
|---------|------|------|------|
| `title` | string | **필수** | 문서/법령 제목 |
| `sub_title` | string | 선택 | 조문/섹션명 필터 (예: `"제3조"`) |
| `limit` | int | 선택 | 최대 반환 건수 (기본값: 50) |

## 도메인(domain) 가이드

### `"law"` — 법령 조문

전세/임대차 관련 법률, 시행령 조문 원문입니다.

**포함 법령:**
- 주택임대차보호법 (법률/시행령)
- 민법
- 부동산등기법
- 공공주택 특별법
- 소득세법 시행령
- 민간임대주택 관련 법률 통합
- 전세사기피해자 관련 법률 통합
- 감정평가 및 감정평가사에 관한 법률 통합

**용도:** 법적 근거가 필요할 때, 조문 원문 확인이 필요할 때

### `"guide"` — HUG 가이드/매뉴얼

HUG에서 제공하는 실무 가이드, FAQ, 매뉴얼 자료입니다. 업로더에서 `카테고리별기본자료`와 `매뉴얼_v2` 디렉토리 모두 `"guide"` 도메인으로 통합 저장됩니다.

**포함 자료 (카테고리별):**
- **(보증가입):** 가입요건, 보증발급불가 대상, 조건변경 등
- **(보증발급):** 전세보증금반환보증 약관, 전세금안심대출보증 약관
- **(보증이행):** 이행청구 절차, 모바일 이행청구, 제출서류, 명도방법, 계약종료/사고사유, 관할업무구역, FAQ
- **(미가입자):** 전세사기 유형별 사례 및 대처방안, 보증금미반환 조회, 상습 채무불이행자 조회
- **(공통):** 임차인을 위한 부동산 계약가이드
- **(매뉴얼):** 안심전세 App 2.0 사용자 매뉴얼, 전세피해지원제도 리플렛, 전세보증금반환보증 주요내용, 전세안심 온라인클래스

**용도:** 실무 절차, FAQ, 가입 조건 등 실용적 정보가 필요할 때

## 사용 전략

### 질문 유형별 도구 선택

| 질문 유형 | 도구 | domain | 예시 |
|-----------|------|--------|------|
| 법적 근거 확인 | `search_hug_docs` | `"law"` | "전세보증금 반환 의무 법적 근거" |
| 실무 절차 질문 | `search_hug_docs` | `"guide"` | "전세보증금반환보증 가입 방법" |
| 종합적 답변 필요 | `search_hug_docs` | `""` (전체) | "전세 계약 만료 후 보증금 못 받으면?" |
| 특정 법조문 확인 | `get_doc_by_title` | - | title="주택임대차보호법", sub_title="제3조" |
| DB에 무슨 문서 있는지 | `list_available_docs` | - | 검색 전 문서 목록 파악 |

### 검색 팁

1. **자연어 질의를 사용하세요.** 키워드보다 문장형이 벡터 검색에 효과적입니다.
   - 나쁨: `"전세 반환"`
   - 좋음: `"전세보증금을 돌려받지 못했을 때 어떻게 해야 하나요?"`

2. **domain 필터를 적극 활용하세요.** 법적 근거가 필요하면 `"law"`, 실무 절차가 필요하면 `"guide"`로 좁혀 검색하면 정확도가 올라갑니다.

3. **title_filter로 범위를 좁히세요.** 특정 법령 내에서만 검색하고 싶을 때 유용합니다.
   - 예: `search_hug_docs(query_text="대항력 요건", domain="law", title_filter="주택임대차보호법")`

4. **2단계 검색 패턴:** 먼저 `search_hug_docs`로 관련 문서를 찾고, 해당 문서의 전체 내용이 필요하면 `get_doc_by_title`로 조회하세요.

5. **limit 조절:** 포괄적인 답변이 필요하면 `limit=10` 이상으로 늘리고, 핵심만 필요하면 기본값 5를 유지하세요.

### 응답 작성 가이드

- 법령 조문을 인용할 때는 **문서제목**과 **조문/섹션명**을 명시하세요.
- `법령링크`가 제공되면 함께 안내하세요.
- 법령(`law`)과 가이드(`guide`)를 조합하여 "법적 근거 + 실무 절차"를 함께 제시하면 가장 유용한 답변이 됩니다.
- 검색 결과가 없거나 부족하면 domain을 바꿔서 재검색하세요.

## 데이터 구축 참고

업로더(`uploader.py`)가 3개 소스 디렉토리에서 JSONL 파일을 읽어 Qdrant에 저장합니다.

| 소스 디렉토리 | domain 값 | 설명 |
|--------------|-----------|------|
| `법령2/` | `"law"` | 법률/시행령 조문 (law_title, sub_title, text, jo_link_url 등) |
| `카테고리별기본자료/` | `"guide"` | HUG 가이드, FAQ, 약관 (title from tags.source, sub_title from title) |
| `매뉴얼_v2/` | `"guide"` | HUG 매뉴얼 (guide와 동일 스키마로 저장) |

- 임베딩 모델: `snowflake-arctic-embed-l-v2.0-ko` (1024차원)
- 리랭커: `bge-reranker-v2-m3-ko`
- 청크: 최대 2000자, 200자 오버랩
- 인덱스: domain(KEYWORD), title(TEXT), category(KEYWORD)
